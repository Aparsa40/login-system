# Login System (v1.0)

A full-stack authentication system using Node.js, Express, MongoDB and JWT.

## Features

- Email & Password Registration
- Secure Password Hashing (bcrypt)
- JWT Authentication
- Login & Signup UI
- Password Confirmation Validation
- Strong Password Rules
- Notification System
- Clean Project Structure

## Tech Stack

- Node.js
- Express
- MongoDB
- JWT
- Vanilla JS (Frontend)

## Setup

1. Clone the repository
git clone your-repo-url

2. Install dependencies

npm install

1. Create `.env` file:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key

1. Run the server

npm run dev

Server runs on:

<http://localhost:5000>
